macOS Sierra开启画中画

发表于 2016-11-11 分类于 macSierra

画中画功能需要Mac 系统为 sierra，浏览器使用 Safari ，网站支持 HTML 播放器，比如 YouTuBe BiliBili。

  1. 在视频播放窗口右键双击，就会出现 进入画中画 的选项，点击后画中画开启。
  2. 随便保存个书签，修改名字为 画中画 ，再修改地址如下： 在视频播放时点击书签就可以。
